Analysis of Election Trends Based on Demographic Data¶

Josh Belew and Catherine Brooks¶

Our github.io¶

Project Goals¶

The goal of this project is to analyze changing voting patterns in the US based on economic, educational and demographic factors. We will be focusing on election results in the past 40 years on a per county basis. We will be using race, religion, gender, and age as our demographic facors. We will be using educational achievement as our educational factor. Lastly, we will be using poverty rate, unemployment rate, and median personal income as our economic factors.

Collaboration Plan¶

We have a set time to meet every Thursday at 1:00p to discuss the work we did that week and what needs to be done the next week, along with any coding we can get done together. We will meet over zoom an additional time every other week to collaborate on code and analysis. We have a shared google drive folder that keeps everything organized and accessible for both of us, along with a private repository branch so we can easily share code.

Table 1: Presidential Election Voting Data by County 1868-2020¶

For the presidential election results we will be using a data set from Harvard Dataverse containing the presidential election results per county from 1868-2020. The dataset can be found here: https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/DGUMFI

In this section we will load the data into pandas. Then we will transform the data by dropping unnecessary columns and caculating the percentage vote for each party. Finally, we will calculate the some statistics and create a graph based on the voting trends in our hometowns.

print('The mean democrat percentage vote in {} county is: {:.2%}'.format(josh_county.lower(), josh_df['percent_democrat'].mean()))
print('The standard deviation of the democrat percentage vote in {} county is: {:.2%} \n'.format(josh_county.lower(), josh_df['percent_democrat'].std()))

# Plot the percentage democratic vote and republican vote over time
